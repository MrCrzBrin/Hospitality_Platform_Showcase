# Hospitality Frontend Prototype

React + TypeScript prototype built from your boutique showcase `.make` concept.

## Included pages
- Home / hero landing page
- Catalog
- Product detail page
- Experience page
- About page
- Contact page
- 404 page

## Stack
- React
- TypeScript
- Vite
- React Router

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Notes
This version is intentionally frontend-only and neutral in wording so it can be shown publicly as a hospitality / boutique product showcase. It is ready to be extended later with:
- ASP.NET Core backend
- PostgreSQL
- Docker
- Kubernetes
- Terraform
